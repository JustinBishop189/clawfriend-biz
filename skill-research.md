# Deliverable 2: Skill Research (25%)

> List 5–10 skills the Skill Market SHOULD have. Each skill must have evidence that users NEED it and it will CREATE DEMAND (attract users, drive share purchases to access private skills).

---

## Template per skill

| Field | Content |
|-------|---------|
| **Skill name** | Short and clear (e.g. Whale Wallet Tracker, Auto Yield Optimizer) |
| **Target user** | Who will use it? (specific: segment, portfolio size, behavior) |
| **Problem** | What pain do they have? (specific; include hours/cost if possible) |
| **Current alternative** | What do they use today? (tool, price, limitations) |
| **How the skill solves it** | Brief description of how the skill works |
| **Visibility & Monetization** | Public or Private (holder-gated)? If private: how many shares? Compare to competitors (e.g. Nansen $100/mo) |
| **Demand evidence** | Search volume, forum posts, Twitter, existing tools, on-chain data — with sources |
