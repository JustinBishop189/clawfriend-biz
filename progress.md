## Step-by-step checklist

### Deliverable 1: Competitive Landscape (25%)

- [x] Find 5–10 competitors (Web3 skill / agent marketplace)
- [x] Per-competitor: name, link, description, metrics, monetization, strengths, weaknesses
- [x] Per-competitor analysis (doing well/poorly, why users choose, lessons)
- [x] Market overview (stage, leaders, room for new entrants)
- [x] Head-to-head comparison table
- [x] Strategic insight & positioning
- [x] Conclusion & positioning statement
- [x] Data sources listed

### Deliverable 2: Skill Research (25%)

- [x] List 5–10 skills the Skill Market should have
- [x] Per skill: name, target user, problem, current alternative
- [x] Per skill: how skill solves it, visibility & monetization (public/private)
- [x] Per skill: demand evidence (search volume, forums, tools, on-chain) with sources
- [x] Source summary (Google Trends, Reddit, Twitter, etc.)

### Deliverable 3: Distribution Plan (40%)

- [ ] Budget overview: $10K month-one allocation (table)
- [ ] At least 3–5 acquisition channels
- [ ] Per channel: why, action plan, timeline (week 1–4), reach, cost, metrics
- [ ] At least 1 organic channel ($0) and 1 paid channel
- [ ] Total paid ≤ $10,000
- [ ] (Bonus) Partnership/collab: partner name, rationale, mutual benefit, actions
- [ ] Overall timeline table (month one)
- [ ] Unit economics (CAC, conversion, targets) if applicable
- [ ] Sanity check: “Could an intern execute this?”

### Supporting outputs

- [ ] [AI Showcase](./ai-showcase/): screenshots or links of AI prompts used for research
- [ ] [Data](./data/): numbers, charts, exports — each with source (URL, date)
- [ ] [README](./README.md): add Gemini Canvas presentation link when ready
- [ ] Web presentation built (Gemini Canvas; no PowerPoint/Google Slides)
- [ ] Repo public; link shared in Telegram
- [ ] All deliverable files pushed to GitHub before presentation
